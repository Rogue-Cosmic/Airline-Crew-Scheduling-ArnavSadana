
# Airline Crew Scheduling — Backtracking Solution

## Overview
This project solves a simplified NP-hard crew scheduling problem using **constraint checking + backtracking**.  
It includes profiling, visualization, and analysis.

## Files
- `crew_scheduling_notebook.ipynb` — Full implementation, profiling & plots  
- `README.md` — Project overview  
- (Optional) `images/` folder for charts  

## How to Run
Install dependencies:
```
pip install matplotlib memory_profiler
```

Run the notebook in JupyterLab or VSCode.
